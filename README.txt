
# WhatsApp Sender - UltraMsg API

مشروع بسيط لإرسال رسائل واتساب وصور باستخدام UltraMsg API.
## الاستخدام:
1. قم برفع الملفات إلى استضافتك (مثلاً InfinityFree داخل مجلد htdocs).
2. افتح `index.html` وأدخل الأرقام والرسالة.
3. يمكن رفع صورة بصيغة PNG.
4. عند الضغط على إرسال سيتم إرسال رسالة أو صورة حسب البيانات المدخلة.

> تأكد من تغيير `token` و `instanceId` في `send.php` بالقيم الخاصة بك من ultramsg.com

تم بواسطة ❤️
